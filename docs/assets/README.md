# Assets

This section covers all game assets required for L4D2 Rage Edition.

## Required Assets

### [Lethal Weapon Assets](ASSETS_LETHAL_WEAPON.md)
Assets required for the Lethal Weapon skill (Saboteur tertiary skill).

### [General Assets & Models](ASSETS_AND_MODELS.md)
Custom models, materials, and other game assets.

## Asset Categories

### Models
- Custom survivor models
- Weapon models
- Effect models

### Materials
- Textures and materials for custom models
- UI elements
- Effect materials

### Sounds
- Music files (see [Music Setup](../features/MUSIC_SETUP_GUIDE.md))
- Player sounds
- Effect sounds

## Installation

1. Copy assets to appropriate directories:
   - Models: `custom/models/`
   - Materials: `custom/materials/`
   - Sounds: `custom/sound/`

2. Ensure file paths match plugin expectations
3. Restart server or reload maps

## Asset Requirements

Most assets are built-in to L4D2. Custom assets are documented in their respective guides.
