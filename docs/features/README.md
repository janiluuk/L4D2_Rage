## Additional Features

- **Self-Revival**: Revive yourself when downed
- **Struggle Escape**: Break free from special infected grabs
- **Ledge Climb**: Pull yourself up from ledges
- **Item Pickup While Incapacitated**: Pick up items while downed
- **Cooldown Notifications**: Visual and audio notifications when skills are ready
- **Class Persistence**: Your class choice is remembered between rounds
- **Music Preferences**: Per-player music settings

## Configuration

Most features can be configured via:
- ConVars (console variables)
- Configuration files in `sourcemod/configs/`
- In-game menus (for players)

See individual feature documentation for specific configuration options.
